# Webpageranking
This repository include visualization and analysis for ranking website. 

## Visualization 
The visualization is done using Bokeh in the ```bokeh\_app``` folder. The folder contents to subfolders and `main.py` file which pass the ```data``` from the data folder to respective tabs by calling different script from the ```scripts``` and present on the dashboard for visualization.
